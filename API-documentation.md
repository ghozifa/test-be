## ENDPOINTS

List of Available Endpoints:

- `GET /initiators`
- `PUT /initiators/submission/:id`
- `GET /participants`
- `GET /participants/:id`
- `PUT /participants/subs/:id`
- `POST /participants`
- `PUT /participants/update/:id`
- `GET /challenges`
- `GET /challenges/:id`
- `GET /challenges/sum/:id`
- `POST /challenges`
- `PUT /challenges/update/:id`
- `DELETE /challenges/:id`
- `GET /admins`
- `POST /admins`
- `POST /admins/login`
- `PUT /admins/submission/:id`
